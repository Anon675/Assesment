# AI/ML Technical Assignment

This repository contains three independent Python scripts addressing the given problem statements.

## Files

- `task1_books_api_sqlite.py`
  - Fetches book data from Open Library (public HTTPS API)
  - Stores records in SQLite
  - Reads and prints stored data

- `task2_scores_api_visualization.py`
  - Fetches data from DummyJSON public API
  - Computes average score
  - Generates a bar chart (`student_scores.png`)

- `task3_csv_to_sqlite.py`
  - Reads local CSV user data
  - Validates and inserts records into SQLite

## Database

- SQLite file: `app.db`
- Created automatically if not present

## CSV

- `users.csv` is expected to be present in the same directory
